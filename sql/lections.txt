 postgresql - Система управления базами данных  (СУБД)
 (Database menedjment systemy DBMS)
 Это ряд программ и инструментов позволяющих создавать БД, управлять ею  и манипулировать данными внутри БД (CRUD)
 POSTGRES - Сама база данных, реляционная, то есть данные хранятся в виде таблиц и таблицы имеют связи между собой (relations)
 SQL - Язык структрированных запросов, декларативный язык программирования, применяемый для создания, модификации и управления данными в реляционной базе данных, управляемой соответствующей системой управления базами данных
 
ubuntu -> sudo -u postgres psql - для открытия СУБД postgresql

CREATE USER _Имя пользователя_ WITH PASSWORD '_Пароль пользователя_'; - создает пользователя с паролем
DROP  USER _Имя пользователя_; - Удаляет пользователя
DROP DATABASE _Название Базы данных_; - удаляет базу данных


Create Database _Название Базы данных_;
Create Table _Название таблицы_
Select - для выведение данные  внутри базы данных (Select * - выводит все, Select _Название Таблицы_ - выводит определённую таблицу)


Типы полей в posgres sql: 
Numeric types:
    a. samallint(2 bytes) -> -32767 to 32767
    b. integer(4 bytes) ->  -2147000 to 2147000
    c. bigint(8 bytes) -> любые числа ...
    d. serial(4 bytes) -> 1-2147000 (autoincrement)
    e. real (4 bytes) -> число с плавающей точкой (десятичные числа)
    f. double precision (8 bytes) -> real но только с двойной точностью

Character types(Строковые типы):
    a. varchar (кол-во 255) -> можем указать макс. символов вручную
    b. char (кол-во  255) -> можем  указать вручную. Если не заполнится то остальные символы заполнится с пробелами
    c. text (не имеет неограниченное кол-во символов).
Boolean:
    boolean (1 bytes) -> True or False

Date - календарная дата (год, месяц. день)

Location - координатная точка -> (x,y) (255, -15)



psql - команда входа в POSTGRES через своего пользователя
\du - выводит список всех пользователей
\dt - список всех таблиц в конкретной Базе данных
\l - список всех баз данных
\c <название базы данных> - команда для подключения  в базу данных
\q - выход из Базы данных
CREATE DATABASE <название>; - команда создания базы данных
CREATE USER <имя пользователя> WITH PASSWORD 'password' ; - команда для создания пользователя
CREATE USER <имя пользователя> WITH SUPERUSER PASSWORD 'password'; - команда для создания суперпользователя 



BETWEEN - SELECT * FROM products WHERE quantity BETWEEN 3 and 8
BETWEEN - Условие от [start: end]
LIKE - Выводит результат который соответствует условию введенному шаблону для строк. Он чувствителен к регистру
ILIKE - Тоже самое но только не чувствителен к регистру
SELECT * FROM products  WHERE name LIKE 'b%'
SELECT * FROM prosucts WHERE name LIKE '% % %'

GROUP BY - Разделяет строки которые возвращают SELECT при этом группируя их по определенному признаку. И теперь для каждой группы можно применить функцию
HAVING - работает также как и WHERE только когда с оператором GROUP BY.




JOIN - Выборка данных из двух таблиц, соединение таблиц
LEFT JOIN - выборка будет содержать все строки первой или левой таблицы 
RIGHT JOIN - Выборка будет содержать все строки второй или правой табоицы
INNER JOIN - выводит данные согласно условию
AVG(), COUNT(), MIN(), MAX(), SUM() - агрегационные функции
